Path Planning for 6 DoF Robotic Arm

This project implement Expansive Space Trees (EST) path planner to plan collision-free motions of an UR5 robot using C++ and ROS Integrated planning simulation with ROS, rviz and MoveIt! the implementation won the Expert Challenge with high success rate for diffcult planning problem on real UR5 robot

ROS2 Introduction and Robot SLAM demo

This tutorial gives an overview of ROS2 (Robotic Operating System) status with an example of SLAM implementation. In this guide, we discuss and evaluate some of its new features by implementing SLAM in ROS2 in simulation and on a real robot and these new features are briefly introduced and some are tested in this implementation.

The project is to use the UR5 robot to complete a "place-and-draw" task using inverse kinematics joint angle control, using differential kinematics rate control, as well as Gradient-based control.

This RC car is armed with a stabilization platform for camera, to enhance the experience of comfortable and flexible video shooting. The car also has a 360 degree rotational robotic arm with four-bar mechanism to enable forward and backward motion, and provide extra degrees of freedom for camera shooting angle.
